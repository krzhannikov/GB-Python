Очень упрощенная база данных абонентов провайдера (ISP). Для простоты все абоненты (таблица customer) - физлица. У каждого абонента есть свой личный менеджер (manager). Также указано, к какому порту какого коммутатора (switch) абонент подключен. Есть таблица задач (task), связанных с абонентом. В таблице задач хранится ссылка (url_crm) на саму задачу в CRM-системе. Есть таблица обращений абонента (cust_request). В ней присутствует ссылка на само обращение (message_url) - это может быть текст из письма или мессенджера, файл записи телефонного разговора, скриншот и т.п. Также обращение может быть связано с задачей (а может и не быть). Ещё есть таблица тарифов (tariff), которые подключены абоненту. Они делятся на три типа услуг - интернет, телефония, телевидение.
